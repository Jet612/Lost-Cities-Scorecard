# Lost Cities · Expedition Ledger

A sleek, modern scorekeeper for the card game **Lost Cities**. Click the cards
each explorer played and the score tallies itself — styled as a 1920s
archaeological field journal.

## Features

- Tap-to-toggle cards for every expedition (3 handshakes + cards 2–10).
- Live per-expedition subtotals and running totals for two players.
- Correct scoring: start at −20, ×(handshakes + 1), +20 bonus at 8+ cards.
- Optional 6th (Purple) expedition.
- Editable player names; progress saved to your browser automatically.

## Develop

```bash
npm install
npm run dev      # start the dev server
npm run build    # production build → dist/
npm run preview  # preview the production build
```

Built with [Vite](https://vitejs.dev/) — vanilla JS, no framework.
